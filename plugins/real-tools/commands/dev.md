> **Let's `/real-tools:dev` this!**

Build, create, and implement complex projects from PRD to final deliverable.

## Usage

```
/real-tools:dev <prd_file_path>
```

## Arguments

- `prd_file_path` (required): Path to Product Requirements Document (markdown file)

If not provided as a command argument, you will be prompted to provide the path.
